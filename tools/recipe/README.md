# 019-recipe-converter｜料理の分量換算ツール

大さじ/小さじ/カップ⇔ml/g の換算と、人数に合わせた材料量の倍率計算ができるツールです。

## 主な機能

- 大さじ(15ml)/小さじ(5ml)/カップ(200ml)/ml ⇔ g 換算（調味料ごとの比重テーブル使用）
- 代表的な調味料・材料12種類（水・砂糖・塩・醤油・油・小麦粉・味噌・みりん・酢・バター・牛乳・はちみつ）の換算表
- 人数換算（基準人数→目的人数の倍率を材料量に乗算）
- 「あくまで目安」である旨をUI・利用規約に明記
- ダーク/ライトモード、PWA対応、完全クライアントサイド・無送信

## ファイル構成

雛形 `factory/product-template/` をベースに、`app.js`へ換算ロジックを実装。
他ファイル（`style.css`/`monetization.js`/`manifest.webmanifest`/`sw.js`/法務ページ）も
本製品向けに設定済み。
